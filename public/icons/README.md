# PWA Icons

This directory should contain the following PNG icons for the PWA:

- `icon-72x72.png` - 72x72 pixels
- `icon-96x96.png` - 96x96 pixels  
- `icon-128x128.png` - 128x128 pixels
- `icon-144x144.png` - 144x144 pixels
- `icon-152x152.png` - 152x152 pixels
- `icon-192x192.png` - 192x192 pixels
- `icon-384x384.png` - 384x384 pixels
- `icon-512x512.png` - 512x512 pixels

## Generating Icons

You can use tools like:
- https://realfavicongenerator.net/
- https://app-manifest.firebaseapp.com/
- `npx pwa-asset-generator` 

Start with a 512x512 or 1024x1024 source image and generate all sizes.

## Icon Requirements

- Use a simple, recognizable design (camera emoji works well)
- Ensure icons have proper padding for "maskable" display
- Use transparent background or solid color background
