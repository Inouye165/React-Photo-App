# PWA Install Demo Videos

Place short looping install demo videos here:

- `install-ios.mp4` → used for iPhone/iPad install instructions in-app
- `install-android.mp4` → used for Android install instructions in-app

Recommended specs:

- Duration: 8-12 seconds
- Aspect: 9:16 portrait (mobile-friendly)
- Codec: H.264 (MP4)
- File size: ideally < 2 MB each
- Audio: optional (UI mutes by default)

Used by:

- `src/pages/ChessGame.tsx` (`InstallDemoModal`)

If a file is missing, the app shows a fallback note and textual steps.
